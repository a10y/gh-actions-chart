Tip: click any bar (or anywhere along its time column) to jump to the exact Actions run on GitHub.
How durations are computed
Queue time = run_started_at − created_at. Run time = updated_at − run_started_at (the workflow run's last update, ≈ completion). Only completed runs have meaningful values.
